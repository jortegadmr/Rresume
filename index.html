Formularios

Para crear formularios utilizamos la etiqueta "form" seguida de la propiedad "action" para indicar la ruta, y el "method" Las Subetiquetas que utilizaremos son; "label" el nombre que está al lado del formulario "input" seguido de "type" Seguido de label podemos añadir un "for" para identificar y que cuando se seleccione el texto vaya al recuadro Después, en input añadimos el "id" que habiamos identificado en for

Para poner el nombre dentro del formulario utilizamos "placeholder" dentro del input

Para desplegar un menu utilizamos la etiqueta "Select" y para las opciones la Subetiqueta "option"

Otras maneras para formularios

  1. Marcar casilla circular: "input type=radio"
  2. Marcar casilla rectangular: "input type=checkbox"
  3. Subir archivo: "input type=file"

Botones: tenemos dos opciones: "input type=submit" y "button type=""" estos botones deben estar dentro de "form"

Tablas

Para crear tablas utilizamos la etiqueta "table", la cual puede estar seguida de sus atributos, ej: "border" Las Subetiquetas son: "tr" para las filas y "td" para las columnas

Imagenes

Para insertar Imagenes se utiliza la etiqueta "img" seguido de sus atributos; que pueden ser: "width" para la anchura, "alt" para el nombre de la imagen y "src" para la ruta de la imagen

Etiqueta SPAM

La etiqueta SPAM sirve para referenciar un texto, es distinto a los enlaces tipo "a" Ejemplo: Lorem ipsum dolor sit, amet consectetur adipisicing elit. Error incidunt beatae delectus ad inventore obcaecati. Consequuntur consequatur ducimus, aliquid dolorum, similique, deleniti non voluptatibus sunt esse natus recusandae ab pariatur.
