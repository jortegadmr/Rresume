Imagenes

Para insertar Imagenes se utiliza la etiqueta "img" seguido de sus atributos; que pueden ser: "width" para la anchura, "alt" para el nombre de la imagen y "src" para la ruta de la imagen


Tablas

Para crear tablas utilizamos la etiqueta "table" que puede estar seguida de sus atributos, ej: "Border"

Las etiquetas utilizadas son "tr" Filas y "td" Columnas

Formularios

Para crear formularios utilizamos la etiqueta "form" seguida de la propiedad "action" para indicar la ruta, y el "method"

Las Subetiquetas que utilizaremos son; "label" el nombre que está al lado del formulario "input" seguido de "type"

Seguido de label podemos añadir un "for" para identificar y que cuando se seleccione el texto vaya al recuadro

Después, en input añadimos el "id" que habiamos identificado en for

Para poner el nombre dentro del formulario utilizamos "placeholder"

Para desplegar un menu utilizamos "select" y para las opciones "option"
